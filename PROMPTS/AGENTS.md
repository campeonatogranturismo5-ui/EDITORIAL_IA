# Instrucciones para `PROMPTS/`

- Mantén un directorio por perfil de agente y versiona cada prompt.
- Deriva el prompt de una ficha de agente aprobada; no inventes responsabilidades.
- Separa instrucciones, ejemplos y datos de prueba.
- No incluyas secretos, datos personales ni contenido de proyectos reales.
- Todo cambio futuro deberá superar pruebas de regresión antes de considerarse activo.
