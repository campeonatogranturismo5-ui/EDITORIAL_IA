# Checklist común de agente

- **Versión:** 1.0
- **Fecha:** 2026-07-23
- **Estado:** `DEFINIDO`
- **Fuente:** `../AGENTS.md` y `../AGENTS/_TEMPLATE/AGENT_TEMPLATE.md`
- **Responsable de aprobación:** Responsable de Calidad

## Antes de ejecutar

- [ ] Identificador, versión, PID y estado comprobados.
- [ ] Entradas completas y autorizadas.
- [ ] Fuentes obligatorias consultadas.
- [ ] Puerta humana aplicable comprobada.
- [ ] Alcance y acciones prohibidas comprendidos.
- [ ] Herramientas autorizadas y secretos protegidos.

## Durante la ejecución

- [ ] Proceso seguido en orden.
- [ ] Decisiones limitadas a la autoridad del agente.
- [ ] Evidencias y referencias conservadas.
- [ ] No se sobrescriben artefactos versionados.
- [ ] Errores y reintentos registrados.

## Antes de entregar

- [ ] Formato exacto de salida cumplido.
- [ ] Criterios de aceptación comprobados.
- [ ] Indicadores calculados cuando corresponda.
- [ ] Artefactos versionados y ubicados correctamente.
- [ ] Contrato de salida completo.
- [ ] Agente siguiente identificado.
- [ ] Registro de auditoría actualizado.
- [ ] Escalado humano realizado si era obligatorio.
- [ ] No se incluyen secretos ni afirmaciones de implementación falsas.
